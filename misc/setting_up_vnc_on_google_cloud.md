# How to set up remote workpace on google compute engine
Google Compute Engine provide 300 USD free credit for the first time user. It is good for people to set up a playground to try different Linux developmenet enviroment

# setup VNC server
The ubuntu instance doesn't come with desktop enviroment. Needs to setup a VNC server. So we could have full functionality through connecting via a VNC client.
